# Communication Guidelines

## Language Preferences
1.  **Chat Interaction**: Always respond in **Simplified Chinese (简体中文)** unless the user explicitly requests English.
2.  **Code & Interface**: Use **Australian English** for all technical artifacts.
    *   **Spelling**: Use `colour`, `organisation`, `licence`, `centre`, `programme`.
    *   **Scope**: This applies to variable names, database fields, function names, comments, and hardcoded UI strings.

## Tone
- Be concise and professional.
- Use direct technical terminology.
